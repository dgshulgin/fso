Библиотека FileSystemObjects (fso) предназначена для использования в надстройках МойОфис Текст и Таблица в качестве инструмента взаимодействия с файловой системой. Библиотека предоставляет следующие возможности:

- Смена рабочего (текущего) каталога;
- Определение рабочего катога;
- Создание каталога;
- Предоставление списка вложенных каталогов и файлов для указанного каталога.

Пример работы с библиотекой на языке Lua:
```lua
local fso = require("fso")
-- печать названия рабочего каталога
print( fso.pwd() )
-- печать списка файлов в каталоге
for idx, file in fso.dir(".") do
    print( file:path() )
end
```

Для сборки необходимы заголовочные файлы и библиотеки Lua 5.4.7.
